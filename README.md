# ⚙️ Mudança de JS puro para React + Vite

Grandes mudanças aconteceram. Uma delas foi a organização das pastas.

Separei o server Front-End do server Back-End, um roda o HTML, o Vite e React.js, outro roda a API do YouTube. Melhor para arrumar e corrigir bugs, erros e organizar códigos, caso tenha alguma outra mudança ou inclusão significativa de alguma nova função ou rotas.

Assim ficou a versão da radio com as mudanças no framework que fiz.

Plugins usados:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc)
